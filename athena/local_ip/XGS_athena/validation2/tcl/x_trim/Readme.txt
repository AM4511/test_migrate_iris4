This folder contains the scripts for generating a simplified validation framework for the x_trim module

Files:
Readme.txt                 : This file
create_modelsim_x_trim.tcl : create the modelsim project
util.tcl                   : Modelsim utility function for running simulations


To start simulation:
1. Start modelsim
2. In the TCL console source $env(IRIS4)/athena/local_ip/XGS_athena/validation2/tcl/x_trim/create_modelsim_x_trim.tcl
3. Press key h for command help menu
